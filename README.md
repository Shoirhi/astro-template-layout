Astroを使ったWebサイトのレイアウトテンプレート。

## 要件
- Astro
- React
- Tailwind CSS
- shadcn/ui

## 機能
- 基本レイアウト（Base Layout）
- SEO
- OGP
- Twitter Card
- サイトマップ
- RSS
- View Transition
- ビューポートが375px未満での表示倍率の縮小（Adjust Viewport）
- スムーススクロール
- Cloudflare Pagesへのデプロイ

## OGPについて
以下のサイトを参照する。
[text](https://ogp.me/)

### OGP画像の大きさ
横1200px縦630pxのJPEGもしくはPNGで作成する。
